# Financial  

Frameworks
- Nuxt 3 (RC)
- Vue 3
- Vuetify 3 (Beta)
- Pinia
- Vitest
- Typescript

Using `setup` to create components

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

Start the development json server on http://localhost:3001

```bash
npm run backend
```
